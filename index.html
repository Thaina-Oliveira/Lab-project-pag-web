Tag Heading

H1: O cabeçalho mais importante, usado para títulos principais em uma página.

H2: Um nível de cabeçalho secundário, adequado para subdivisões de tópicos na página.

H3: Um nível de cabeçalho mais baixo, útil para subseções dentro das subdivisões.

H4 a H6: Níveis ainda mais baixos de cabeçalhos, para hierarquias de informações mais detalhadas.

H1 a H6: Todos esses elementos são blocos de cabeçalho e geralmente vêm com uma mudança de estilo visual, como tamanho de fonte e negrito, para denotar sua importância hierárquica.

Listas

Ul: Representa uma lista não ordenada, onde os itens não têm uma sequência específica. Geralmente é mostrada com marcadores (pontos, círculos, quadrados, etc.).

Ol: Representa uma lista ordenada, onde os itens seguem uma sequência numérica ou alfabética. Cada item é automaticamente numerado.

Li: É usado dentro de listas (ul ou ol) para definir cada item da lista. Contém o conteúdo que você deseja listar, seja texto, imagens ou outros elementos.

Tag P

Voltar

P: A tag pé usada para criar parágrafos de texto em HTML. Ela é usada para estruturar o conteúdo em blocos de texto separados, indicando ao navegador que deve haver uma quebra de linha antes e depois do parágrafo.

Tag Mark

Mark: A tag mark é usada para destacar parte do texto dentro de um parágrafo ou outro elemento de texto. Geralmente, o texto destacado é realçado com uma cor de fundo amarela para chamar a atenção do leitor.

Tag Small

Small: A tag small é usada para definir texto em um tamanho de fonte menor do que o texto circundante. Ela é frequentemente usada para indicar que o texto é menos importante, como notas de rodapé, legendas, informações legais ou detalhes secundários. Essa tag ajuda a melhorar a formatação do conteúdo, destacando informações menos proeminentes sem comprometer a legibilidade do texto principal.

Tag U

U: A tag U é usada para sublinhar o texto, indicando que é uma parte de conteúdo que deve ser exibida com uma linha horizontal embaixo. No entanto, o uso dessa tag é considerado obsoleto e não é mais recomendado para formatação de texto, pois o sublinhado é frequentemente associado a links na web.

Tag Strong

Strong: A tag strong é usada para dar ênfase forte ao texto, indicando que é uma parte de conteúdo importante ou significativa. Ela geralmente é interpretada pelo navegador como um texto mais enfático, frequentemente exibido em negrito. No entanto, a intenção principal da tag Strong é transmitir importância semântica ao conteúdo, em vez de apenas aplicar formatação visual.

Voltar

Tag Hr

Hr: A tag Hr é usada para criar uma linha horizontal, geralmente para separar seções de conteúdo em uma página da web. Essa linha horizontal é frequentemente usada como uma divisão visual, como uma linha de separação entre parágrafos, tópicos ou seções.

A tag Hr não requer uma tag de fechamento, pois é uma tag vazia. Ela cria uma linha horizontal que se estende por toda a largura do seu contêiner pai.

Tags Sub e Sup

Sub: A tag sub é usada para definir texto como subscrito, o que significa que o texto é posicionado um pouco abaixo da linha de base normal. Isso é frequentemente usado para representar índices, fórmulas químicas ou outras notações em que o texto menor é colocado abaixo do nível principal do texto.

H2O é a fórmula da água.

Sup: A tag sub é usada para definir texto como sobrescrito, o que significa que o texto é posicionado um pouco acima da linha de base normal. Isso é frequentemente usado para representar expoentes, notas de rodapé ou outras notações em que o texto menor é colocado acima do nível principal do texto.

Tag Blockquote

Blockquote: A tag blockquote é usada para criar um bloco de citação que é tipicamente recuado e estilizado de maneira diferente do restante do conteúdo. Ela é frequentemente usada para destacar uma citação mais longa ou um bloco de texto que foi retirado de outra fonte.
